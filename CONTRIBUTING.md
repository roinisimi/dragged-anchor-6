# Contributing to Dragged Anchor

Thank you for contributing to our anchor management system!

## Submission Guidelines

1. **Fork the repository**
2. **Create your anchor configuration** in the `anchors/` directory
3. **Test locally** if possible
4. **Submit a Pull Request**

## File Naming Convention

- Anchor configs: `anchor-<vessel-name>.txt`
- Config files: `<config-name>.conf`

## What Happens After Submission

When you submit a PR:
1. Our CI/CD pipeline validates the configuration
2. Specs are uploaded to Pastebin for review by our engineering team
3. Once approved, your changes are merged

## Questions?

Open an issue and we'll help you out!
